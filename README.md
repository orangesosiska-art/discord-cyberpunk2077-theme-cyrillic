
# Cyberpunk 2077 Theme - Cyrillic Script Support

Changes the font of the Cyberpunk theme to support all letters of the Cyrillic script. 

Меняет шрифт Киберпак темы на другой для поддержки кирилицы.

Змінює шрифт Кіберканк теми на інший для підтримки кирилиці.

Original font - Rajdhani

Fork's font - Tektur

## Website

Website related to the original theme with configuration tool and FAQ - [cyberpunk.pbielanin.pl](https://cyberpunk.pbielanin.pl)
## Installation

Download the theme from the repo and copy files to your Better Discord/Legcord/Vencord/Shelter Themes folder. 
Legcord, Vencord and Shelter have special theme installation tools in the Settings.
